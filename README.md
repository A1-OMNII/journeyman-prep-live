# journeyman-prep-live\n\nThis repository contains a Streamlit frontend (streamlit_app.py) and a FastAPI backend (backend/app.py) for license issuance and verification via Stripe webhooks.\n\n## Quick local run\n\nPrereqs: Python 3.8+\n\n1. Create a virtualenv and install deps:\n\n```bash\npip install -r requirements.txt\n```\n\n2. Run backend locally:\n\n```bash\nexport APP_BASE_URL=http://localhost:8000\nuvicorn backend.app:app --reload\n```\n\n3. Run the Streamlit app in a separate shell:\n\n```bash\nstreamlit run streamlit_app.py\n```\n\n4. For Stripe webhook testing locally, use stripe cli to forward events or deploy backend to a public host first.\n\n## Env vars (set in deployment)\n\n- STRIPE_SECRET_KEY\n- STRIPE_WEBHOOK_SECRET (set after you create webhook in Stripe)\n- SENDGRID_API_KEY (optional, for automatic license emails)\n- APP_BASE_URL (public backend URL)\n- SECRET_KEY (admin endpoints)\n- DATABASE_PATH (optional, defaults to licenses.db)\n\n## Deployment\n\nRecommended: Deploy Streamlit app (Streamlit Cloud) and backend to Render or Heroku. After backend is deployed add a Stripe webhook for `checkout.session.completed` pointing to `https://<your-backend>/webhook` and copy the webhook secret into STRIPE_WEBHOOK_SECRET.\n
